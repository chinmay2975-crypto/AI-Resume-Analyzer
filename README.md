# AI Resume Analyzer using NLP

This project analyzes resumes and compares them with job descriptions using Natural Language Processing (NLP).

## 🚀 Features
- Extracts text from PDF resumes
- Identifies key skills
- Matches resume with job description
- Calculates match score using TF-IDF and cosine similarity
- Highlights missing skills

## 🛠️ Tech Stack
- Python
- PyPDF2
- Scikit-learn
- Matplotlib

## 📊 How it Works
1. Resume is parsed from PDF
2. Text is processed using NLP techniques
3. TF-IDF converts text into vectors
4. Cosine similarity computes match score
5. Skills are extracted and compared
